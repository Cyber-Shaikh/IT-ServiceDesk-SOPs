## Objective
To provide effective support for corporate iOS devices while ensuring security, compliance, and minimal user disruption.

## Scope
Applicable to iPhone and iPad devices used by enterprise users and managed through Microsoft Intune.

## Supported Activities
- New iOS device setup and user onboarding
- Email and application configuration
- Device compliance and policy checks
- Device replacement and re-enrollment
- Lost or stolen device handling

## Initial Checks
1. Verify device ownership (corporate or BYOD)
2. Confirm user identity and device assignment
3. Check device enrollment status in Intune
4. Verify network and Apple ID configuration

## Common Issues
- Email not syncing on iPhone/iPad
- Device marked as non-compliant
- Applications not installing
- Passcode or access-related issues

## Troubleshooting Steps
1. Validate device compliance status in Intune
2. Sync device policies
3. Reinstall or update required applications
4. Guide user through basic device checks
5. Re-enroll device if required

## Lost or Stolen Device Handling
1. Verify incident details with the user
2. Mark device as lost or wipe device remotely
3. Block corporate access if required
4. Guide user on replacement and re-enrollment

## Outcome
Ensure secure access to corporate resources while maintaining device compliance and user productivity. Document actions taken in the ITSM tool.
