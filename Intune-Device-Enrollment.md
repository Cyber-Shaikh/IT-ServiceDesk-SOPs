## Objective
To enroll and manage corporate endpoint devices using Microsoft Intune to ensure security, compliance, and centralized management.

## Scope
Applicable to Windows 10/11 and iOS devices used in an enterprise environment.

## Enrollment Prerequisites
- Device assigned to the user
- User account available in Azure AD (Entra ID)
- Valid Microsoft Intune license
- Internet connectivity

## Enrollment Process
1. Verify user and device eligibility
2. Initiate enrollment using Company Portal
3. Authenticate user with corporate credentials
4. Apply configuration and security policies
5. Validate successful enrollment and compliance status

## Post-Enrollment Checks
- Device appears in Intune portal
- Compliance policies applied
- Required applications installed
- Device marked as compliant

## Common Issues
- Enrollment failures
- Device not appearing in Intune
- Compliance policy errors

## Resolution
Troubleshoot enrollment issues, re-initiate enrollment if required, and confirm device compliance.

## Outcome
Ensure devices are securely managed, compliant with organizational policies, and ready for enterprise use.
