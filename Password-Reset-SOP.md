## Objective
To securely reset user passwords while ensuring identity verification and minimal business disruption.

## Scope
Applicable to enterprise users accessing corporate systems via Active Directory or Azure AD.

## Pre-Checks
- Verify user identity using approved verification methods
- Confirm account status (active, locked, or expired)
- Check for related MFA or access issues

## Password Reset Process
1. Validate user request and identity
2. Reset password in Active Directory or Azure AD
3. Enforce password change at next login if required
4. Unlock account if locked
5. Confirm successful login with the user

## Post-Reset Actions
- Advise user on password security best practices
- Update incident or request ticket with actions taken

## Outcome
Ensure secure and timely restoration of user access while maintaining compliance with security policies.
