## Objective
To manage user access requests securely and efficiently in an enterprise environment.

## Scope
This SOP covers access provisioning, modification, and deprovisioning activities.

## Standard Activities
- New user access enablement
- Role-based access changes
- Group membership updates
- Access removal during offboarding

## Process
1. Validate access request and approvals
2. Update access in Azure AD or Active Directory
3. Verify access with the user
4. Document changes in the ITSM tool

## Outcome
Ensure users have appropriate access while maintaining security and audit compliance.
