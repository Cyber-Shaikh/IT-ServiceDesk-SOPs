# IT Service Desk SOPs & Documentation

This repository contains standard operating procedures (SOPs) and troubleshooting documentation based on real-world enterprise IT service desk experience.

The documentation is aligned with ITIL-based support processes and focuses on common IT support scenarios involving Microsoft 365, Azure AD (Entra ID), endpoint devices, and identity management.

## Coverage Areas
- User Access & Identity Management
- MFA Troubleshooting
- Endpoint & Device Support (Windows 10/11, iOS)
- Intune Device Enrollment & Compliance
- VPN & Connectivity Issues
- Incident Handling & Escalation Workflows

## Purpose
- Improve first-contact resolution
- Reduce repeat incidents through structured documentation
- Support consistent and reliable IT service delivery

> Note: All documentation is generalized and does not contain any client-specific or confidential information.
